# Interactive Roller Coaster Track Simulator and Physics Engine

An interactive, lower-level graphics environment designed to dynamically plot, trace, and virtualize kinematic paths over complex compound rails. Built from scratch in **C#** and rendered using native GDI+ **WinForms Graphics**, this software replaces commercial game engines with raw mathematical implementations to compute vector mechanics, track transformations, and path step sequences in real time.

## Algorithmic & Mathematical Architecture

The framework bypasses modern wrapped rendering layers, calculating every cart movement vector through explicit backend mathematical functions:

### 1. Vector Path Segment DDA Interpolation
To trace linear track spans smoothly, the engine uses a custom **Digital Differential Analyzer (DDA)** class. It determines precise point increments by evaluating coordinate deltas ($\Delta x$, $\Delta y$) and slopes ($m$), systematically calculating optimal micro-steps based on axis dominance:

$$\text{Dominant } \Delta x \implies C_x \pm= \text{speed}, \quad C_y += m \cdot \text{speed}$$

### 2. Parametric Bézier Curves from Scratch
To create flexible hills and valley spans, the application uses higher-order **Bézier Splines**. The system calculates Bernstein polynomial combinations manually by running discrete mathematical loops for Factorials ($n!$) and Binomial Coefficients ($C(n, i)$):

$$C(n, i) = \frac{n!}{i!(n - i)!}$$

$$P(t) = \sum_{i=0}^{n} C(n, i) \cdot (1 - t)^{n - i} \cdot t^i \cdot P_i$$

### 3. Circular Polar Projection Loops
Loop-the-loops and circular track curves are modeled using explicit polar trigonometry functions. By tracking radial steps ($\theta$) across radian vectors, the model derives smooth arc transitions and tracks exact next-point targets for cart positions:

$$X = R \cdot \cos(\theta) + X_c, \quad Y = R \cdot \sin(\theta) + Y_c$$

### 4. Translation and Rotation Matrix Transformations
To adjust car orientations dynamically as they move along slopes, the system runs coordinate transformations around custom reference points, mapping unaligned coordinate offsets across angular rotation vectors:

$$X_{new} = X \cdot \cos(\phi) - Y \cdot \sin(\phi)$$

$$Y_{new} = X \cdot \sin(\phi) + Y \cdot \cos(\phi)$$

## Key Features
- **Pure Math Implementations:** Bypasses engine wrappers to execute custom parametric mathematics, binomial combinatorics, and discrete transformations directly inside text scripts.
- **Dynamic Track Assembly:** Allows real-time track design by connecting linear lines, trigonometric arcs, and free-form Bézier curves on an infinite canvas.
- **Interactive Mouse Manipulation:** Implements live click-and-drag bounding box lookups to modify control points and reshape track paths on the fly.
- **Double-Buffered Rendering Layer:** Minimizes frame flickering during real-time physics tracking by processing updates within off-screen bitmap buffers before rendering.
- **Scrolled Canvas Virtualization:** Built-in viewport scrolling fields handle out-of-bounds tracking, shifting view matrices automatically as the simulated cart advances.
